# .github — padrões da organização Vertex Devs

Este repositório especial define os padrões herdados por **todos os repos da org**:

| Arquivo | O que faz |
|---|---|
| `profile/README.md` | Perfil público da organização |
| `ENGINEERING.md` | Manual de engenharia da Vertex (fonte da verdade) |
| `CONTRIBUTING.md` | Fluxo de branch, commit e PR |
| `SECURITY.md` | Política de segredos e reporte de vulnerabilidades |
| `PULL_REQUEST_TEMPLATE.md` | Template padrão de PR (herdado por todos os repos) |
| `ISSUE_TEMPLATE/` | Templates de bug, feature e pendência |
| `workflow-templates/` | Workflows de CI reutilizáveis (aba Actions → New workflow) |

Repos individuais podem sobrescrever qualquer um destes arquivos criando a própria versão.

Novos projetos: use o repo template [`template-vertex`](https://github.com/vertexdevs-hq/template-vertex).
